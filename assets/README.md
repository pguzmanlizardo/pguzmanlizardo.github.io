# Assets Directory

This directory contains all media and document files for the website.

## Structure

- `/photos/` - Field work photos for the "Photos from the Field" section
- `cv.pdf` - Your curriculum vitae (place your CV PDF here)

## Instructions

### Adding Your Portrait
1. Place your portrait photos in this directory:
   - `portrait.jpg` - Front image (shown by default)
   - `portrait2.jpg` - Back image (shown when clicked)
2. Recommended dimensions: 600x800px (3:4 aspect ratio)
3. File format: JPG or PNG
4. Keep file size under 200KB for optimal loading
5. Clicking the portrait will trigger a 3D flip animation between the two images

### Adding Your CV
1. Place your CV PDF file in this directory
2. Name it `cv.pdf` or update the links in `index.html` to match your filename

### Adding Photos
1. Place your field work photos in the `/photos/` subdirectory
2. Update the image sources in the "Photos from the Field" section of `index.html`
3. Recommended image formats: JPG or PNG
4. Recommended maximum dimensions: 1920x1080px for optimal web performance
5. Consider compressing images before uploading to improve page load times

### Adding Abstract Images
1. Place two images to display next to your JMP abstract:
   - `abstract-image-1.jpg` - First image (shown by default)
   - `abstract-image-2.jpg` - Second image (accessible via navigation)
2. Recommended aspect ratio: 4:3 (e.g., 800x600px)
3. File format: JPG or PNG
4. Keep file size under 300KB each
5. These could be charts, graphs, maps, or photos related to your research

## Tips
- Use descriptive filenames for photos (e.g., `fieldwork-amazon-2024.jpg`)
- Keep file sizes reasonable (aim for under 500KB per photo)
- Maintain consistent aspect ratios for a cleaner gallery appearance
- For the portrait, a professional headshot or formal photo works best
